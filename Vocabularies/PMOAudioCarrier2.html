PMO Audio Carrier Vocabulary

  • 1/2 track
    • USE : half track
  • 1/4 track
    • USE : quarter track
  • 12-track
    • UF : twelve-track
    • DEF : Magnetic tape divided into 12 audio tracks that run parallel with each other, in which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • 16-track
    • UF : sixteen-track
    • DEF : Magnetic tape divided into 16 audio tracks that run parallel with each other, in which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • 45 recording
    • DEF : Colloquial term representing 7" diameter vinyl audio discs with a playing speed of 45 rpm. The term does not necessarily describe the carrier accurately.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • 78 recording
    • DEF : Colloquial term representing audio discs from the 78 rpm disc era with playing speeds ranging from around 60 rpm to 130 rpm. The term does not necessarily describe the carrier accurately.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • 8-track
    • UF : eight-track
    • DEF : Magnetic tape divided into 8 audio tracks that run parallel with each other, on which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • CD Audio
    • DEF : Optical discs that use the standard compact disc format for audio. They are 12 cm (4 3/4 in.) in diameter, and encoded as 2-channel signed 16-bit Linear PCM sampled at 44,100 Hz.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • DAT
    • DEF : Sony's proprietary Digital Audio Tape format, characterized by magnetic media enclosed in a protective shell, with the dimensions of 8 x 6 cm, 4 mm tape or 2 7/8 x 2 1/8 in., 1/8 in. tape.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1007
  • DVD Audio
    • DEF : DVD optical discs that use the the DVD audio format. Different bit depth/sampling rate/channel combinations can be used on a single disc.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • DVD Video
    • DEF : DVD optical discs that use the DVD format to store digital video. From 2003, it has been the dominant consumer video format.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1060
  • eight-track
    • USE : 8-track
  • Enhanced CD Audio
    • DEF :
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • EP record
    • UF : extended play record
    • DEF : A music disc that contains more tracks than a single but not as many as on a typical album/LP/disc.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • extended play record
    • USE : EP record
  • full track
    • DEF : Magnetic tape with 1 audio track on which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • half track
    • UF : 1/2 track
    • DEF : Magnetic tape divided into 2 audio tracks that run parallel with each other, on which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • hybrid SACD
    • DEF : An SACD that also is also encoded with a PCM audio layer readable by compact disc players.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • laserdisc
    • DEF : Home video format, with a standard diameter of 30 cm (12 in.)
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • LP
    • DEF : Colloquial term representing 12-inch diameter vinyl audio discs with a playing speed of 33 1/3 rpm. The term does not necessarily accurately describe the carrier.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • organ roll
    • DEF : An audio roll used to operate a player organ or reproducing organ.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1006
  • piano roll
    • DEF : An audio roll used to operate a player piano or reproducing piano.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1006
  • quarter track
    • UF : 1/4 track
    • DEF : Magnetic tape divided into 4 audio tracks that run parallel with each other, in which sound and other electro-acoustic signals are captured.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1008
  • SACD
    • UF : Super Audio Compact Disc
    • DEF : Optical audio disc with more channels and longer playing time than a conventional CD.
    • TT : http://rdaregistry.info/termList/RDACarrierType/1004
  • sixteen-track
    • USE : 16-track
  • Super Audio Compact Disc
    • USE : SACD
  • twelve-track
    • USE : 12-track
